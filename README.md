## 📖 项目简介

OfferForge 是一款专为**技术面试准备**的**分布式刷题平台**，提供算法题库、学习路线、模拟面试等功能。通过分布式架构支持高并发，支持多人同时在线刷题和竞赛。

## 🚀 快速开始

### 环境要求

- JDK 8+
- MySQL 5.7+
- Maven 3.6+

### 数据库配置

```sql
CREATE DATABASE offerforge DEFAULT CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
SOURCE sql/init.sql;
```

### 编译运行

```bash
# 编译项目
mvn clean package

# 运行项目
java -jar target/offerforge-1.0.0.jar

# 或开发模式运行
mvn spring-boot:run
```

### Docker 部署

```bash
# 构建镜像
docker build -t offerforge .

# 运行容器
docker run -d -p 8080:8080 offerforge
```

## ❓ 常见问题

### Q: 如何开始刷题？

> 建议按照学习路径从第一阶段开始，先掌握基础知识和简单题目，逐步提升难度。

### Q: 题目有解答吗？

> 大部分题目都有题解和讨论区，可以查看其他用户的解法，也可以分享自己的思路。

### Q: 如何提高效率？

> 建议每天固定刷题时间，配合学习路线图系统推进，不要只刷简单题。
